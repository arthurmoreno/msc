\justifying
{
Neste trabalho aplicamos métodos computacionais conhecidos para estudar as dinâmicas dos processos de infecção de duas espécies virais, o HIV e o HSV, o método do Runge-Kutta de 4º Ordem e um modelo de Autômato Celular. Utilizamos a Mecânica Estatística, especificamente o conceito de Entropia para analisar propriedades específicas destas dinâmicas. O comportamento da Entropia, no caso do HIV caracteriza as diferentes fases finais do processo de infecção. No caso do HSV a entropia máxima reflete características diferentes para os dois tipos de úlceras, as ameboides e as dendríticas.
}
